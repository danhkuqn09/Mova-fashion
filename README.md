# E-commerce Backend (Laravel)

## Description
This is a backend system for an e-commerce application built using Laravel.

## Features
- Authentication (Laravel Sanctum)
- Role-based access control (Admin/User)
- Product management
- Order management
- Payment integration (Momo)

## Tech Stack
- PHP (Laravel)
- MySQL
- RESTful API

## My Role
- Backend Developer
- Responsible for API development, database design, and business logic

## Frontend
Frontend is handled separately by another team member.

## How to run
- Clone repository
- Run `composer install`
- Setup `.env`
- Run `php artisan serve`
