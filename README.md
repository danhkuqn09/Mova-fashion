# E-commerce Backend (Laravel)

## Description
This is a backend system for an e-commerce application built using Laravel.

## Features
- Authentication (Laravel Sanctum)
- Role-based access control (Admin/User)
- Product management
- Order management
- Payment integration (Momo)

## API Highlights
- Authentication API (login, register)
- Product API (CRUD, search, filter)
- Order API (create order, update status)

## Tech Stack
- PHP (Laravel)
- MySQL
- RESTful API

## Project Structure
- Controller: Handle request/response
- Service: Business logic
- Model: Database interaction

## How to run
- Clone repository
- Run `composer install`
- Copy `.env.example` to `.env`
- Configure database
- Run `php artisan serve`
